Tests MathJax

Lors de la compilation des fichiers Markdown avec Pandoc, il suffit d’ajouter --mathjax pour que la page web génère de belles équations grâce à MathJax. Quelques tests :

  • Test 1 : \(a^2 + b^2 = c^2\).
  • Test 2 : \[ E = mc^2 \]
  • Test 3 : \[ \begin{align} x + y &= 1 \\ 2x - y &= 3 \end{align} \]
  • Test 4 : \[ \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} \]
  • Test 5 : \(\forall \epsilon, \, \exists \delta\).
  • Test 6 : \(\mathcal{F}:\mathcal{A}\to \mathcal{B}\).
